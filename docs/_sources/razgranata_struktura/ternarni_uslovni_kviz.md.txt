## Питање 2

Нека су `a` и `b` целобројне променљиве типа `int`. Које су наредбe
еквивалентне наредби `if (a > 0) b = 1; else b = 0;`? Штиклирај тачне одговоре.

- `(a > 0) ? (b = 1) : (b = 0);` **ТАЧНО**
- `a > 0 ? (b = 1) : (b = 0);` **ТАЧНО**
- `(a > 0) ? b = 1 : b = 0;`
- `b = a > 0 ? 1 : 0;` **ТАЧНО**

## Питање 3

Нека је `a` целобројна променљива типа `int`. Којe наредбe проверавају и
исписују да ли је вредност целобројне променљиве `a` непаран или паран број?
Штиклирај тачне одговоре.

- `a % 2 == 1 ? printf("neparan") : printf("paran");` **ТАЧНО**
- `a % 2 != 1 ? printf("neparan") : printf("paran");`
- `a % 2 < 0 || a % 2 > 0 ? printf("neparan") : printf("paran");` **ТАЧНО**
- `a & 1 ? printf("neparan") : printf("paran");` **ТАЧНО**
