## Питање 2

```c
___________________ ? ___________________ : ___________________
```

```{mchoice}
:answer1: На празне линије треба уписати (a > 0), (b = 1) и (b = 0);
:answer2: На празне линије треба уписати a > 0, (b = 1) и (b = 0);
:answer3: На празне линије треба уписати (a > 0), b = 1 и b = 0;
:answer4: На празне линије треба уписати b = a > 0, 1 и 0;
:correct: 1,2,4

Нека су a и b целобројне променљиве типа int. Које опеанде треба да упишеш на
датим линијама да би формирао наредбу која ће бити еквивалента наредби
if (a > 0) b = 1; else b = 0;?
```

## Питање 3

```c
___________________ ? printf("neparan") : printf("paran");
```

```{mchoice}
:answer1: a % 2 == 1
:answer2: a % 2 != 1
:answer3: a % 2 < 0 || a % 2 > 0
:answer4: a & 1
:correct: 1,3,4

Нека је a целобројна променљива типа int. Које услове можеш написати у датом
коду на линији да би се проверило и на стандардном излазу исписало да ли је
вредност a непаран или паран број?
```
