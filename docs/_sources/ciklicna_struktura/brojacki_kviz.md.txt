# Квиз: наредба бројачког циклуса

## Питање 1

Која ће бити вредност променљиве `sum` након извршења следећег кода у
програмском језику C?

```c
int sum = 10;
for (int i = 5; i <= 10; i++) {
    sum += i;
}
```

- 20
- 25
