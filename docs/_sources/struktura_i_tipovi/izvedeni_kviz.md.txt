# Квиз: изведени и набројиви типови

## Питање 1

Која је сврха коришћења кључне речи `enum` у програмском језику C?

- дефинисање скупа података са коначним бројем вредности **ТАЧНО**
- дефинисање реалних константних податка
- дефинисање низова карактера
- дефинисање синонима за постојеће типове података

## Питање 3

У дефиницији енумерације `enum e { a, b, c};`, коју вредност има променљива
`a`?

- вредност није дефинисана

- $1$
- $a$

## Питање 4

Дефинисан је изведени тип `typedef enum { a, b, c } е;`. Програмер треба да
декларише променљиву `prom` типом `e` и додели јој другу вредност из енум
скупа. На који начин то може да уради? Штиклиран тачне одговоре.

- `prom = е[2];`
- `e prom = 2;`
- `e prom = b;` **ТАЧНО**
- `e prom = 1;` **ТАЧНО**

## Питање 5

Шта ће се исписати на стандардном излазу након извршавања следећег програма?

```c
#include <stdio.h>

int main(void)
{
    typedef enum { po, ut, sr, ce, pe, su, ne } d;
    d v1 = su, v2 = ne;
    printf("Dani vikenda: %d, %d", su, ne);
    return 0;
}
```

- `Dani vikenda: 5, 6` **ТАЧНО**
- `Dani vikenda: 6, 7`
- `Dani vikenda: su, ne`
- Исписаће се порука о грешки.
