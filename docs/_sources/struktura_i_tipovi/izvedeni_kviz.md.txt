## Питање 5

```c
#include <stdio.h>

int main(void)
{
    typedef enum { po, ut, sr, ce, pe, su, ne } d;
    d v1 = su, v2 = ne;
    printf("Dani vikenda %d, %d", su, ne);
    return 0;
}
```

```{mchoice}
:answer1: Dani vikenda 5, 6
:answer2: Dani vikenda 6, 7
:answer3: Dani vikenda su, ne
:answer4: Исписаће се порука о грешци.
:correct: 1

Шта ће се исписати на стандардном излазу након извршавања следећег програма?
```
