## Питање 9

```c
1. printf("%d.%d.%d.", &d, &m, &g);
2. printf("%d.%d.%d.", d, m, g);
3. printf("%d%d%d", d.m.g.);
4. printf("%d", d.m.g.);
```

```{mchoice}
:answer1: Наредба 1.
:answer2: Наредба 2.
:answer3: Наредба 3.
:answer4: Наредба 4.
:correct: 2

Алиса је рођена 20.10.2008. године. Ако су иницијализоване следеће променљиве:
int d = 20, m = 10, g = 2008;, која ће наредба исписати Алисин датум рођења
20.10.2008. на стандардни излаз?
```

## Питање 10

```c
#include <stdio.h>

int main(void)
{
    int broj;
    scanf("%d", &broj);
    printf("%+05d", broj);
    return 0;
}
```

```{mchoice}
:answer1: Исписаће се 5
:answer2: Исписаће се 00005
:answer3: Исписаће се +0005
:answer4: Исписаће се 000005
:correct: 3

Шта ће се исписати на стандардном излазу, ако је корисник након покретања
датог програма унео број 5?
```
