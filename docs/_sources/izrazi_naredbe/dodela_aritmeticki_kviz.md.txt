## Питање 3

Шта ће се исписати на стандардном излазу након покретања следећег програма?

```c
#include <stdio.h>

int main(void)
{
    double res = 6 + 5 / 4 + 3;
    printf("%.2lf", res);
    return 0;
}
```

- 5.75
- 6.71
- 10.00 **ТАЧНО**
- 10.25

## Питање 5

Шта ће се исписати на стандардном излазу након покретања следећег програма?

```c
#include <stdio.h>

int main(void)
{
    int a = 5, b = 4, c = 3;
    double res = (double)a * b / c;
    printf("%.2lf", res);
    return 0;
}
```

- 5.00
- 6.00
- 6.67 **ТАЧНО**
- 7.00
