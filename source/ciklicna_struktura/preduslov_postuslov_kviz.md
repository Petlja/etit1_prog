## Питање 3

```c
unsigned num = 11, s = 0;
while (num != 0) {
    if (num & 1) s++;
    num >>= 1;
}
printf("%u", s);
```

```{mchoice}
:answer1: ...исписаће број цифара у бинарном запису броја `num`.
:answer2: ...исписаће број цифара у декадном запису броја `num`.
:answer3: ...исписаће број јединица у декадном запису броја `num`.
:answer4: ...исписаће број јединица у бинарном запису броја `num`.
:correct: 4
```

## Питање 4

```c
int broj;
do {
    scanf("%d", &broj);
    if (broj & 1)
        printf("%d\n", broj);
} while (broj & 1);
```

```{mchoice}
:answer1: ...исписује унете бројеве, све док се не унесе јединица.
:answer2: ...исписује унете јединице, све док се не унесе број различит од један.
:answer3: ...исписује унете позитивне бројеве, све док се не унесе негативан.
:answer4: ...исписује унете непарне бројеве, све док се не унесе парни.
:correct: 4

Следећи кôд у програмком језику C...
```

## Питање 5

```c
for (int i = 5; i < n; i++) s += i;
```
