## Питање 2

```c
int x = 0 != 1 < 2;
```

```{mchoice}
:answer1: израз је недефинисан јер редослед извршавања операција није дефинисан заградама.
:answer2: променљивој x додељује се вредност логичке истине 1
:answer3: променљивој x додељује се вредност логичке неистине 0
:answer4: променљивој x додељује се вредност 0, а остатак израза се игнорише.
:correct: 2

Који ће исказ бити тачан након извршења дате наредбе?
```

## Питање 3

```c
int num;
scanf("%d", &num);
int x = 0 != 1 || 2 != num;
```

```{mchoice}
:answer1: израз ће увек бити недефинисан јер редослед операција није дефинисан заградама.
:answer2: променљивој x биће додељенo 0 или 1 у зависности вредности num.
:answer3: без обзира на вредност num, променљивој x биће додељено 1.
:answer4: без обзира на вредност num, променљивој x биће додељено 0.
:correct: 3

Који ће исказ бити тачан након извршења датих наредби?
```
