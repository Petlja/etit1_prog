# Квиз: Наредба гранања

## Питање 1

```c
int a;
scanf("%d", &a);
if (a = 1) printf("DA");
else printf("NE");
```

```{mchoice}
:answer1: Без обзира на унету вредност `a`, увек ће се исписати `DA`.
:answer2: Без обзира на унету вредност `a`, увек ће се исписати `NE`.
:answer3: У зависности од унете вредности `a`, исписаће се `DA` или `NE`.
:answer4: Компајлер ће јавити грешку јер `printf()` функције нису записане у новом реду.
:correct: 1

Шта ће се исписати на стандардном излазу након извршења датог кода?
```
