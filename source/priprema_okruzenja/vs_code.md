# Visual Studio Code едитор и GCC

:::{admonition} Напомена
:class: warning

Ова лекција намењена је само оним ученицима који, из неког разлога, нису у
могућности да инсталирају и користе интегрисано развојно окружење Visual
Studio. Ако си већ инсталирао Visual Studio, онда нема потребе да инсталираш
и Visual Studio Code едитор и GCC сет алата!
:::

Једна од алтернатива интегрисаном развојном окружењу Visual Studio је Visual
Studio Code едитор и GCC сет алата. Остале алтернативе, Code::Blocks и DevC++
су прилично застареле по дизајну и функционалности и одавно се не користе у
софтверским компанијама. Ако из неког разлога не можеш или не желиш да користиш
Visual Studio, у наставку ће бити објашњен детаљан поступак преузимања,
инсталације и конфигурације Visual Studio Code едитора за рад са GCC сетом
алата.

Овај поступак је мало компликованији од претходних, па пажљиво прати следеће
кораке.

## Преузимање и инсталација Visual Studio Code едитора

Преузимање и инсталација Visual Studio Code едитора прилично је једноставна и
интуитивна. Visual Studio Code едитор је потпуно бесплатaн, чак и за
комерцијалну употребу. Да би га преузео са интернета, потребно је посетиш веб
страницу
[code.visualstudio.com/Download](https://code.visualstudio.com/Download)
и одабереш адекватну верзију. Ако немаш администраторске привилегије на
рачунару, одабери `User Installer`, односно, ако имаш администраторске
привилегије, одабери `System Installer` како би едитор био доступан свим
корисницима рачунара.

```{image} images/vscode0.png
:width: 100%
:align: center
:alt: code.visualstudio.com
```

Покрени преузети инсталер и прихвати услове коришћења:

```{image} images/vscode1.png
:width: 70%

:alt: Visual Studio Code инсталација
```

У овом кораку можеш променити фолдер у којем ће се Visual Studio Code
инсталирати. За тим нема потребе, па је довољно да оставиш подразумевани фолдер
и кликнеш `Next`.

```{image} images/vscode2.png
:width: 70%

:alt: Visual Studio Code инсталација
```

У овом кораку можеш променити име фолдера у Start менију у којем ће бити
креирана пречица до Visual Studio Code едитора. Ни за тим нема потребе, па је
довољно да оставиш подразумевано име и кликнеш `Next`.

```{image} images/vscode3.png
:width: 70%
:align: center
:alt: Visual Studio Code инсталација
```

У овом кораку можеш одабрати да ли желиш пречицу на десктопу штиклирањем опције
`Create a desktop icon` и да ли у Windows Explorer контекстном менију желиш
ставку `Open with Code` када кликнеш десним кликом на фајл или директоријум
штиклирањем опција
`Add "Open with Code" action to Windows Explorer file context menu` и
`Add "Open with Code" action to Windows Explorer directory context menu`.
Следеће две опције су већ штиклиране,
`Register Code as an editor for supported file types` и
`Add to PATH (requires shell restart)`, а тичу се асоцијације Visual Studio
Code едитора са подржаним форматима фајлова и додавања Visual Studio Code
едитора у системску путању како би се могао позивати из било ког директоријума.
Препоручено је да ове опције оставиш штиклираним. Након одабира жељених опција
кликни `Next`.

задовољан, кликни `Install`, а у супротном можеш кликнути `<Back` и направити
измене.

```{image} images/vscode7.png

:alt: Visual Studio Code инсталација
```

Када се едитор стартује по први пут у менију са леве стране кликни на икону
`Extensions` или притисни на тастатури `CTRL`, `SHIFT` и `X`. У пољу за

## Преузимање и инсталација GCC сета алата

Након инсталације едитора, неопходно је да инсталираш C компајлер, линкер и
дебагер. GNU колекција компајлера и алата за Windows дистрибуира се кроз Cygwin
и MinGW пројекте, па ћеш у овој лекцији научити да инсталираш пакет mingw-w64
који подржава програмирање у програмском језику C на 32-битним и 64-битним
верзијама оперативног система Windows.

За инсталацију MinGW користи се [MSYS2](https://www.msys2.org/) исталер.

```{image} images/msys0.png
:width: 100%
:align: center
:alt: msys2.org
```

Тренутно актуелну верзију MSYS2 инсталера можеш преузети кликом на линк поред
текста `1. Download the installer`. Инсталирај MSYS2 као на сликама испод.

```{image} images/msys1.png
:width: 70%
:align: center
:alt: msys2.org
```

Кликни `Next`.

```{image} images/msys2.png
:width: 70%
:align: center
:alt: MSYS2 инсталација
```

Немој мењати инсталациону путању `C:\msys64` и кликни `Next`.

```{image} images/msys3.png
:width: 70%
:align: center
:alt: MSYS2 инсталација
```

Кликни `Next`.

```{image} images/msys4.png
:width: 70%
:align: center
:alt: MSYS2 инсталација
```

Искључи опцију `Run MSYS2 now.` и кликни `Next`.

```{image} images/msys5.png
:width: 70%
:align: center
:alt: MSYS2 инсталација
```

Овим си завршио инсталацију MSYS2.

директоријума. Покрени Windows Settings и у пољу за претрагу унеси
`Edit environment variables for your account`. У секцији `User variables`
одабери `Path` и кликни `Edit`.
