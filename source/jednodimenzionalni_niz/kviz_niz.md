## Питање 2

```{mchoice}
:answer1: Индекси елемената низа су увек целобројног типа int.
:answer2: Индекси елемената низа могу бити целобројног или реалног типа.
:answer3: Индекси елемената низа су увек реалног типа double.
:answer4: Индекси елемената низа немају одређен тип.
:correct: 1

Који је од наведених исказа тачан?
```

## Питање 3

```{fitb}
:answer: "0,1,9"

Попуни празнине цифрама (не речима). Ако једнодимензионални низ има 10
елемената, онда је индекс првог елемента низа |blank|, другог елемента |blank|,
... и десетог елемента |blank|.
```

## Питање 5

```{mchoice}
:answer1: 4
:answer2: 10
:answer3: 40
:answer4: 80
:correct: 3

Ако је декларисан низ int arr[10];, онда ће извршавањем наредбе
int n = sizeof(arr); променљива n добити вредност...
```
